User Documentation

GetJob is a skill job recommender app, wherein users can register with their education details and skills and get job recommendations. The webapp has three interfaces for the users to interact with.

Registering as a Job seeker

To use the services of the webapp, a user must have an account registered with the webapp. To register,

  1. On your browser, go to getjob.apps.pcfdev.in/signup
  2. Fill your personal details like email, name, age, gender and city of residence.
  3. Click on Save & next, after filling all the necessary details.
  4. Fill your Educational details, then Save & next.
  5. Fill in your Skills and Work Experience.
  6. Click on sign up button to submit.

Note: You cannot set your own password, the webapp generates a secure 10 character alphanumeric password and sends it to your email, the password is not stored on the server.
Instead passhash of the password is stored, so if you loose your password, you'll have to reset it.

You'll be redirected to the login page.

Logging in

If you received your login credentials, or you already have an account, you can login to the webapp.

  1. On your computer go to getjob.apps.pcfdev.in/login
  2. The login form is displayed, enter your email that you used for registration.
  3. Enter the 10 character password that you received via email, in the password box.
  4. click on login button to login.

If you lost your password, you can reset your password here.

Resetting your password

The secure password is emailed to the user, if the password is lost the user needs to reset it.

  1. On your browser, go to getjob.apps.pcfdev.in/forgotpassword
  2. On the reset password page, enter your registered email.
  3. Click on reset passsword button.
  4. If you have an account, a new password will be sent to your email.

NOTE: If you don't have an account, a message is displayed on the login page, informing the same. You can register here.

User Dashboard

After logging to the webapp, the user's dashboard is displayed.

Job Recommendations

Job recommendations are displayed on the job page, whenever user logs into the webapp, these job recommendations are updated everytime.

Using the search bar

Users can use the search bar to look for jobs, which are displayed based on the user's profile.To search for a job,

  1. Click on the category drop down and select a category from skill, job or location.
  2. Enter the search text in the search box.
  3. Click on Search. To find jobs.

Main page

This is the landing page of the website.

Our Employers

The carousel lists our top employers, clicking on the images displays the jobs offered by the organization.

Job Gallery

Jobs by roles

Displays various roles offered by organisations, clicking on it, displays job openings for the role offered by various organisations.

Jobs by organisations

Displays various organisations offering jobs in the portal, clicking on it, displays all the job roles offered by the organisation.

Learning Resources

Third party Learning resources are displayed at the end of the page, which are update based on user's profile.

Chat bot

User's can interact with the chatbot and get job recommendations for their skills.

  1. The chat bot is visible on all pages except the landing page of the website.
  2. To chat with the bot, click on at the bottom of the page.
  3. The bot can help you find
    • Job recommendations for skills
    • Job application status
    • Api Documentation
    • Website Documentation
    • Provide Learning Resources
    • Greet and reply to nrml msgs
  4. The webpages and results can be accessed by clicking the buttons returned as msg from the chat.

GetJOB's Job Search api is a powerful api, exposed to the programmer through a simple HTTP interface. Query parameters are sent via HTTP GET query and search results, errors and diagnostic information are returned as an JSON document to ther user. The same api interface serves the chatbot of the site.

URL :

http://getjob.apps.pcfdev.in/api/v1.0/jobs

Examples :

curl is free and open source command line software used to transfer data. Curl comes preinstalled on most of the devices. We will use curl to access the api.

curl -u {api token}:unused -i -X GET http://getjob.apps.pcfdev.in/v1.0/jobs?search=value

curl -u {api token}:unused -i -X GET http://getjobwebapp.apps.pcfdev.in/api/v1.0/jobs?search=appliedjobs

curl -u {api token}:unused -i -X GET http://getjobwebapp.apps.pcfdev.in/api/v1.0/jobs?apply=uniquejobid

For more details refer to api Documentation page.

Profile page

Profile page displays user's details which user has entered while regestering,this also displays user's application status.

Applied Jobs

Clicking on show more under job status in profile, takes the user to the applied jobs where the status of all the applied jobs is displayed.

Applying For a job

User's can use all the three interfaces(using webapp, chatbot and API) to get job recommendations and to apply for a job.

Using the webapp

The user can choose jobs from

  • Job recommendations on jobs page
  • Search results of search bar
  • From Job gallery on landing page

Using the chatbot

  1. Hello Andrea
  2. I want job recommendations
  3. My skills are C,C++,Python.

Using the REST API of the webapp

Refer to documentation to use the api to apply for jobs.

All these result in a Apply button, clicking on which an application is sent to the employer along with your resume. The status of the application's can be checked from the user's profile page.

Status of Job Application

After applying for a job, an email confirmation is sent to the user from getjob, if you are one of the qualified applicant's the recruiters of the organization will call you to set up a virtual interview.

The status of the application can be viewed from the profile page of the user or you can ask the chat bot.

Logging out

Click on Log out from the navigation bar to log out of the webapp.
