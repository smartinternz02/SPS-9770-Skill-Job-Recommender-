Job Search API

This document describes the Job Search API.

Summary

GetJOB's Job Search api is a powerful api, exposed to the programmer through a simple HTTP interface. Query parameters are sent via HTTP GET query and search results, errors and diagnostic information are returned as an JSON document to ther user. The same api interface serves the chatbot of the site.

URL :

http://getjob.apps.pcfdev.in/api/v1.0/jobs

Examples :

curl is free and open source command line software used to transfer data. Curl comes preinstalled on most of the devices. We will use curl to access the api.

curl -u {api token}:unused -i -X GET http://getjob.apps.pcfdev.in/v1.0/jobs?search=value
curl -u {api token}:unused -i -X GET http://getjobwebapp.apps.pcfdev.in/api/v1.0/jobs?search=appliedjobs
curl -u {api token}:unused -i -X GET http://getjobwebapp.apps.pcfdev.in/api/v1.0/jobs?apply=uniquejobid

Authorization

A user is authorized by their api token, the api token is sent along with the api request, which the system verifies, and return

search data to the user. Your api token is available on the top of this page.


Input Parameters

 
Parameter Format

The input parameter must be sent using HTTP GET. Care should be taken to URL-encode the parameter correctly. Malformed parameter will either be incorrectly interpreted and or give rise to unexpected results. The parameter used is a multi-valued parameter and doesn't have a name.

Query parameters
Name Description Example Results in
search Location api/v1.0/jobs?search=Hyderabad The list of available jobs in Hyderabad
search skill api/v1.0/jobs?search=python The list of available jobs requiring python skill
search position api/v1.0/jobs?search=full-stack-developer The list of available jobs with position as Full-stack-developer
search Organisation api/v1.0/jobs?search=Infosys The list of available jobs at Infosys
search Applied jobs api/v1.0/jobs?search=appliedjobs The list of applied jobs and their status
apply Apply for a job api/v1.0/jobs?apply=2349 Apply to the job that has the unique id 2349
